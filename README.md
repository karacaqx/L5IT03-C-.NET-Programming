# Employee Management System - Student Project

A simple web-based employee management system built as a university assignment using HTML, CSS, JavaScript, and Bootstrap.

## Project Overview

This is a student project created for the Web Development course. The system allows administrators to manage employee information with basic CRUD (Create, Read, Update, Delete) operations.

## Features

### Core Functionality
- **Login System**: Department-based authentication
- **Employee Search**: Find employees by ID
- **Employee Management**: View and edit employee details
- **Dashboard**: Simple statistics and navigation
- **Responsive Design**: Works on desktop and mobile devices

### Technical Features
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Data Storage**: JSON-based local storage (for demonstration)
- **Authentication**: Session-based login system
- **Responsive Design**: Mobile-first approach

## File Structure

```
Website/
├── index.html              # Login page
├── dashboard.html           # Main dashboard
├── employee-management.html # Employee CRUD operations
├── README.md               # Project documentation
└── run-server.bat          # Quick server startup script
```

## How to Run

### Method 1: Python HTTP Server
1. Open terminal/command prompt in project folder
2. Run: `python -m http.server 3000`
3. Open browser: http://localhost:3000/index.html

### Method 2: Use the batch file
1. Double-click `run-server.bat`
2. Browser will open automatically
